# Abhishek Kumar Dutta | Personal Portfolio

A sleek, modern, and highly responsive personal portfolio website built to showcase my projects, skills, and creative work as a Web Developer, Filmmaker, and Music Producer. Designed with a premium **Glassmorphism** aesthetic, fluid micro-animations, and a dynamic particle background.

**[🚀 View Live Demo](https://abhishekkumardutta.netlify.app/)**

## ✨ Features

- **Glassmorphism Design:** Beautiful frosted glass UI cards with depth layers and translucent elements.
- **Dynamic Animations:** Floating background orbs, an interactive canvas particle system, and smooth scroll reveals.
- **Fully Responsive:** Flawless experience across desktop, tablet, and mobile devices with a custom mobile navigation menu.
- **Typewriter Effect:** Animated hero text showcasing multiple roles dynamically.
- **Functional Contact Form:** Integrated with Formspree for direct email delivery without a backend.
- **Optimized Performance:** Vanilla HTML/CSS/JS for lightning-fast load times and smooth rendering.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom properties (variables), Flexbox/Grid layouts, keyframe animations, and backdrop-filters.
- **JavaScript (Vanilla):** DOM manipulation, intersection observers, canvas animations, and form handling.
- **Lucide Icons:** Beautiful, consistent, and lightweight vector icons.

## 🚀 Getting Started

To run this project locally, simply clone the repository and open the `index.html` file in your browser. No build steps or package managers are required!

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- (Optional) VS Code with the **Live Server** extension for a better development experience.
